# Engage 2028

Conversational UI mockup for AI-powered customer engagement operations — "ChatGPT for running customer engagement operations."

## Live Demo

Open `index.html` in a browser, or use GitHub Pages once published.

## Features

- **Business agent sidebar** — Collections, Onboarding, Journey Review, Proactive Support, Lead Acquisition
- **Conversational interface** — ChatGPT-style UX with inline journey cards, execution updates, and reports
- **Per-agent context bar** — Lightweight operational awareness, not a dashboard
- **Explainability** — "Why did you do this?" capability built into the flow

## Usage

```bash
open index.html
```

Click any agent on the left to see a sample conversation with embedded journey cards and agent responses.
